# Práctica WEB-API/Node.js/MongoDB


### Instalación

	$ git clone https://github.com/siono/4-KC-Practica-NodeJS.git
	$ cd nodepop
	$ npm install

### Arrancar MongoDB

	$ bin/mongod --dbpath ./data/db --directoryperdb
      
### Arrancar servidor
	$ npm start
      
### Arrancar la base de datos
	$ npm run installDB
